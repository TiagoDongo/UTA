Linguagem Fundamental da Web: HTML

Escrito por Laura Jones Segunda-feira 21 de Junho 2027

Todos os sites e aplicativos da Web modernos são criados usando três tecnologias fundamentais: HTML, CSS e Javascript

Estas são as linguagens da Web

Neste post, vamos nos concentrar em HTML. Vamos aprender o que é tudo HTML sobre, e por que você também deve aprender.

O que é HTML?

HTML significa HyperTextMarkup Language. É uma linguagem de marcação que os desenvolvedores web utilizam para estruturar e descrever o conteúdo de uma página web. (não é uma linguagem de programação).

HTML consiste em elementos que descrevem diferentes tipos de conteúdo: parágrafos, links, títulos, imagens, vídeos, etc. Navegadores da Web entender HTML e renderizar código HTML como sites.

Em HTML, cada element é formado por 3 partes:

  1. Tag de abertura
  2. Tag de fechamento
  3. O conteúdo do elemento

Você pode aprender mais em MDN Web Docs.

Porquê aprender HTML?

Existem inúmeras razões para aprender a linguagem fundamental da Web.Aqui estão 5 deles:

  • Ser capaz de usar a linguagem de desenvolvimento web fundamental
  • Para criar sites bonitos à mão em vez de depender de ferramentas como Wordpress ou Wix
  • Para construir aplicações Web
  • Impressionar seus amigos
  • Divertir
