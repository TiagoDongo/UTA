A linguagem básica da Web: HTML

Postado por Laura Jones na segunda-feira, 21 de junho de 2027

Todos os sites e aplicativos da web modernos são construídos usando três tecnologias fundamentais: HTML, CSS e JavaScript. Estas são as linguagens da web.

Neste post, vamos nos concentrar em HTML. Vamos aprender o que é HTML e por que você também deve aprendê-lo.

O que é HTML?

HTML significa HyperText Markup Language. É uma linguagem de marcação que os desenvolvedores da Web usam para estruturar e descrever o conteúdo de uma página da Web (não uma linguagem de programação).

HTML consiste em elementos que descrevem diferentes tipos de conteúdo: parágrafos, links, cabeçalhos, imagens, vídeo, etc. Os navegadores da Web entendem HTML e processam o código HTML como sites.

Em HTML, cada elemento é composto de 3 partes:

  • A etiqueta de abertura
  • A etiqueta de fechamento
  • O elemento real
